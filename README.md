# Swiggy SQL Data Analysis

## Project Overview

This project analyzes a Swiggy restaurant dataset using SQL to extract business insights and understand restaurant performance, customer ratings, cuisine trends, and city-wise distribution.

## Tools Used

* SQL
* MySQL

## Key Analysis Performed

* Top-rated restaurants
* City-wise restaurant distribution
* Cuisine analysis
* Restaurant rating trends
* Aggregation and grouping
* Subqueries for business insights

## SQL Concepts Used

* SELECT
* WHERE
* ORDER BY
* GROUP BY
* HAVING
* Aggregate Functions
* Subqueries

## Business Insights

* Identified top-performing restaurants.
* Analyzed cuisine popularity across cities.
* Evaluated restaurant ratings and customer preferences.
* Generated data-driven insights for decision making.
